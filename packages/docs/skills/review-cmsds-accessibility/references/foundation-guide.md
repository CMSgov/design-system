# Review Accessibility for CMS Design System Foundations

The CMS Design System currently provides accessibility guidance for the following foundational topics:

- Typography
- Spacing
- Layout Grid

## Locate the foundation guidance

For CMS Design System versions `v18.0.0` and later, use the following base path:

`node_modules/@cmsgov/design-system/dist/docs/foundation/`

If `dist/docs` is not present (CMS Design System versions earlier than `v18.0.0`), use the following base path instead:

`https://design.cms.gov/foundation/`

Append one of the following paths:

- **Typography**
  - `typography/body/llms.txt`
  - `typography/links/llms.txt`
  - `typography/lists/llms.txt`
- **Spacing** → `spacing/llms.txt`
- **Layout Grid** → `layout-grid/layout-grid/llms.txt`

## Review the Accessibility guidance

Within the relevant page's `llms.txt`, locate the **Accessibility** section.

Review:

- General accessibility guidance.
- Accessibility testing guidance.

Use these sections as the primary source of truth when reviewing accessibility for the selected foundation topic.
