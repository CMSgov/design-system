# Review Accessibility for CMS Design System Page Layouts

The CMS Design System currently provides accessibility guidance for the **Article Page** layout.

## Locate the layout guidance

Read:

`node_modules/@cmsgov/design-system/dist/docs/layouts/article-page/llms.txt`

If `dist/docs` is not present (CMS Design System versions earlier than `v18.0.0`), use:

`https://design.cms.gov/layouts/article-page/llms.txt`

## Review the Accessibility guidance

Within the layout's `llms.txt`, locate the **Accessibility** section.

Review:

- General accessibility guidance.
- Accessibility testing guidance.

Use these sections as the primary source of truth when reviewing the accessibility of pages that follow the Article Page layout.
