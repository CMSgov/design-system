# Review Accessibility for CMS Design System Patterns

The CMS Design System currently provides accessibility guidance for the following patterns:

- 404 Page
- Forms
- Utility Actions

## Locate the pattern guidance

For CMS Design System versions `v18.0.0` and later, use the following base path:

`node_modules/@cmsgov/design-system/dist/docs/patterns/`

If `dist/docs` is not present (CMS Design System versions earlier than `v18.0.0`), use the following base path instead:

`https://design.cms.gov/patterns/`

Append one of the following paths:

- **404 Page** → `404-page-guidance/llms.txt`
- **Forms**
  - `forms/error-validation/llms.txt`
  - `forms/form-review/llms.txt`
- **Utility Actions** → `utility-actions/llms.txt`

## Review the Accessibility guidance

Within the pattern's `llms.txt`, locate the **Accessibility** section.

Review:

- General accessibility guidance.
- Accessibility testing guidance.

Use these sections as the primary source of truth when reviewing the accessibility of the pattern.
