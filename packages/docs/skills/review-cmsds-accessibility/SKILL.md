# Review CMSDS Accessibility

Use this skill to review the accessibility of CMS Design System components, patterns, layouts, typography, and related guidance according to official CMS Design System documentation.

## Determine the relevant guidance

If you are reviewing a CMS Design System component (for example, Alert, Button, Badge, or Drawer), refer to:

`references/component-guide.md`

If you are reviewing a CMS Design System pattern (for example, Forms or the 404 Page), refer to:

`references/pattern-guide.md`

If you are reviewing the CMS Design System Article Page layout, refer to:

`references/layout-guide.md`

If you are reviewing foundational design guidance, such as typography, spacing, or layout grid, refer to:

`references/foundation-guide.md`

For general CMS Design System accessibility guidance—including supported WCAG conformance, accessibility principles, and links to Section 508 resources—refer to:

`node_modules/@cmsgov/design-system/dist/docs/guidelines/accessibility/llms.txt`

If `dist/docs` is not present (CMS Design System versions earlier than `v18.0.0`), use remote documentation:

https://design.cms.gov/guidelines/accessibility/llms.txt
