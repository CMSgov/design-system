## Guidance

- Content authors must ensure that phone numbers are shown as links on the appropriate device. Users on desktop devices cannot use a link to initiate a phone call, so we suggest adding links only for mobile devices users. Developers are responsible for adding this device dependent functionality.
- If the phone number has a letter format, then only the number format of the phone number would be linked.
- If the phone number only shows up as letters then the letter format of the phone number would be linked.
- All numbers need to be followed by details for Teletypewriter (TTY) users.
