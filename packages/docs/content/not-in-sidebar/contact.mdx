## <GithubIcon /> Github

If you're looking to contribute to the CMS Design System check out the [contributing guidelines](https://github.com/CMSgov/design-system/blob/main/CONTRIBUTING.md) to learn how to contribute back to the codebase. 

<a href="https://github.com/CMSgov/design-system/issues" target="_blank">Create a Github issue</a>


## <JiraIcon /> Jira

If you're looking to file an official request to our backlog use Jira. 

**Note:** You'll need specific permissions to access Jira. If you don't have access use [GitHub](https://github.com/CMSgov/design-system/issues) instead. 

<a href="https://jira.cms.gov/secure/RapidBoard.jspa?rapidView=3359&projectKey=WNMGDS&view=planning.nodetail&issueLimit=100" target="_blank">Create a Jira ticket</a>


## <SlackIcon /> Slack

Reach out to the team directly in CMS Slack by messaging the #cms-design-system channel.

**Note:** You'll need specific permissions to access CMS Slack.

<a href="https://cmsgov.slack.com/archives/CHH0381RD" target="_blank">Send us a Slack message</a>
