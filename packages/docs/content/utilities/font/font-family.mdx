title: Font Family

Change an element's typeface using a font family utility.

**Format**: `.ds-u-sans`

<EmbeddedExample>
  <section class="example-list">
    <article>
      <code class="ds-u-margin-bottom--1">.ds-u-sans</code>
      <p class="ds-u-sans ds-u-font-size--4xl ds-u-margin-y--0">
        The quick brown fox jumps over the lazy dog
      </p>
    </article>
  </section>
</EmbeddedExample>

## Guidance

### Style customization

The following CSS variables can be overridden to theme the font family utility:

- `--font-family-sans`
- `--font-family-serif`
