title: Font style
---
import loremIpsumGenerator from '../../../src/helpers/loremIpsumGenerator';

Change an element's `font-style` property using the font style utility.

**Format**: `.ds-u-font-style--[normal|italic]`

 <EmbeddedExample id="1">
  <section class="example-list">
    <article>
      <code class="ds-u-margin-bottom--1">.ds-u-font-style--normal</code>
      <div class="ds-u-font-style--normal">{loremIpsumGenerator('m')}</div>
    </article>
    <article>
      <code class="ds-u-margin-bottom--1">.ds-u-font-style--italic</code>
      <div class="ds-u-font-style--italic">{loremIpsumGenerator('m')}</div>
    </article>
  </section>
</EmbeddedExample>
