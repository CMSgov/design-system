### Changing icon color

Icon components support a few built-in color options that can be applied by using one of the following utility classes.

If you need a different color for an icon, you can pass a custom class to an icon component through the `class` prop. Then, in your application CSS, set the `color` prop.

| Color   | CSS utility Class                     | Example                                                                                                                     |
| ------- | ------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| Primary | <code>ds-c-icon-color--primary</code> | <div class="ds-u-text-align--center"><CloseIcon class='ds-c-icon-color--primary' /></div>                                   |
| Inverse | <code>ds-c-icon-color--inverse</code> | <div class="ds-base--inverse ds-u-padding--05 ds-u-text-align--center"><CloseIcon class='ds-c-icon-color--inverse' /></div> |
| Error   | <code>ds-c-icon-color--error</code>   | <div class="ds-u-text-align--center"><CloseIcon class='ds-c-icon-color--error' /></div>                                     |
| Warn    | <code>ds-c-icon-color--warn</code>    | <div class="ds-u-text-align--center"><CloseIcon class='ds-c-icon-color--warn' /></div>                                      |
| Success | <code>ds-c-icon-color--success</code> | <div class="ds-u-text-align--center"><CloseIcon class='ds-c-icon-color--success' /> </div>                                  |

### Changing icon size

Icon components use `ems` for their height and width and can thereby be sized by changing the font-size of the SVG element.

It is recommended that the [font-size utilities](/utilities/font/font-size/) be used.

| Size        | Example                                   |
| ----------- | ----------------------------------------- |
| Small       | <CloseIcon class='ds-u-font-size--sm' />  |
| Default     | <CloseIcon />                             |
| Large       | <CloseIcon class='ds-u-font-size--lg' />  |
| Extra Large | <CloseIcon class='ds-u-font-size--xl' />  |
| 2x Large    | <CloseIcon class='ds-u-font-size--2xl' /> |
| 3x Large    | <CloseIcon class='ds-u-font-size--3xl' /> |
| 4x Large    | <CloseIcon class='ds-u-font-size--4xl' /> |
| 5x Large    | <CloseIcon class='ds-u-font-size--5xl' /> |

## Accessibility

The `<SvgIcon>` component has built-in accessibility features including:

- a built-in `title` attribute. For any specific exported icon, the `title` attribute is defaulted, but can be overwritten.
- `focusable="false"` which prevents the icon from being focusable
- `role="img"` which tells assistive technologies that the icon's purpose is as an image
- optional `description` attribute which can be used for a more detailed explanation of the icon's contents. Meant to be used to provide more detail than just the `title`
- an available`ariaHidden` attribute which can be used if the icon's context is described elsewhere. For example, if the icon is part of a button with text, the text may provide the text description instead of the icon component.
