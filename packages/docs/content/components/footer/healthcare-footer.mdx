## Accessibility

The Healthcare.gov footer has been tested against, and meets, the WCAG 2.1 standard for accessibility.

- The footer includes the Healthcare.gov logo. [Read more about testing the logo’s accessibility](/components/logos/healthcare-logo/?theme=healthcare).

### Accessibility testing

#### Screen reader testing

When using a screen reader:
- Screen reader announces each interactive element with its type (link or button) and content.
- Screen reader announces the logo as image and link, and the link’s destination.

#### Keyboard testing

When navigating with a keyboard:
- Use the `Tab` key to move forward and `Shift` + `Tab` to move backward
through interactive elements.
- Confirm that focus is clearly visible on each interactive element.
