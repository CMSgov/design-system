---
title: Using LLMs.txt
---

Our documentation now includes an `llms.txt` file at the root of the site [design.cms.gov/llms.txt](https://design.cms.gov/llms.txt).

The `llms.txt` file is a lightweight, Markdown-based index designed to help AI tools identify and retrieve the most relevant Design System documentation. It provides a structured entry point to key pages and guidance, making it easier for language models to access accurate, up-to-date CMS Design System context.

Teams who use AI-assisted development tools can configure their tools to fetch and incorporate this file into their workflow. Integration approaches will vary depending on the tool or workflow being used.
