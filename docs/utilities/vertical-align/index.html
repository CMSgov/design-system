Vertical align

utilities/vertical-align.scss:4

Use the vertical align utility on inline-level elements (inline and inline-block) and on table cells.

Format: ds-u-valign--[VALUE]

A Top alignment
B Middle alignment
C Bottom alignment
Code snippet
<table>
