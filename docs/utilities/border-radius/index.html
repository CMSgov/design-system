    Border radius

    View source file
    UsageGuidance

    Apply rounded corners to an element with the border radius utility.

    • ds-u-radius applies the default radius (3px)
    • ds-u-radius--pill
    • ds-u-radius--circle can make a circle when combined with a fixed height and width.
